## 🙄 What is WRYD?

![WRYD Logo](https://github.com/air4sss/wryd_movie/blob/main/logo_rmbg.jpg)

**WRYD**는 "What aRe You Doing?"의 약자로, 사용자에게 친근하게 다가가며 현재 하고 있는 일을 물어보는 의미를 담고 있습니다. 이를 통해 사용자가 원하는 정보를 제공하고, 더 나은 사용자 경험을 제공하고자 합니다.
